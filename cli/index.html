CLI

swag cli

swag init

--generalInfo value

Go file path in which 'swagger general API Info' is written (default: "main.go")

--dir value

Directory you want to parse (default: "./")

---swagger value

Output the swagger conf for json and yaml (default: "./docs/swagger")
